# Disputes

Information about whether to file a dispute, how to file, what evidence helps, what happens next, dispute status, and Branch role in the process.

## Before filing

Check whether the issue is a pending authorization, merchant refund, lost card, or unauthorized charge before opening a dispute.

## After filing

Branch can help collect information, provide updates, and communicate next steps through the dispute process.

{% hint style="info" icon="life-ring" %}
Still need help? Use Branch app chat when you can access the app. If you cannot log in, use Help Center chat and include the relevant account, card, payout, or transaction context.
{% endhint %}
