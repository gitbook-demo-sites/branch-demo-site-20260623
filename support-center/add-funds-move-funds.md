# Add funds and move funds

Information about outbound transfers, adding funds to Branch, ATM access, cash back, ACH transfers, debit card links, and connected accounts.

## Adding funds

Move money into Branch through direct deposit, ACH, check deposit, cash deposit, or supported gig-payment connections.

## Outbound transfers

Transfer funds out of Branch using eligible instant transfer and ACH options; verify debit card or bank details when prompted.

{% hint style="info" icon="life-ring" %}
Still need help? Use Branch app chat when you can access the app. If you cannot log in, use Help Center chat and include the relevant account, card, payout, or transaction context.
{% endhint %}
