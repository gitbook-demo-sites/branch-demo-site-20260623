# Security best practices

Keep Branch account access safe with passcode hygiene, verified support channels, and careful handling of verification codes.

## Protect your account

Use a unique passcode, keep your phone number current, and avoid sharing sensitive details outside approved Branch support channels.

## When to escalate

Escalate immediately if a card is lost, a charge is unrecognized, or account contact information changed unexpectedly.

{% hint style="info" icon="life-ring" %}
Still need help? Use Branch app chat when you can access the app. If you cannot log in, use Help Center chat and include the relevant account, card, payout, or transaction context.
{% endhint %}
