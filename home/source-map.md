# Source map

This first draft combines three inputs:

* Branch public support portal: worker-facing article categories, common questions, and support routing.
* Branch main website: brand language around faster workforce payments, flexible payouts, embedded finance, tips, mileage, paycard, direct deposit, and business audiences.
* Evolve demo site pattern: wide homepage, AI assistant prompts, persona toggles, picked-for-you adaptive blocks, and card-based routing.

{% hint style="info" icon="wand-magic-sparkles" %}
The Support Center section is designed as a migration-style first pass. The GitBook website import was also started against the source portal so the long tail can be pulled in directly where the importer can access it.
{% endhint %}
