# Branch API Reference

This API section uses a dummy OpenAPI specification to show how Branch could publish partner and embedded-finance APIs in GitBook.

{% hint style="warning" icon="flask" %}
This is a demo spec, not a production Branch API contract.
{% endhint %}

## Base URLs

| Environment | URL |
| --- | --- |
| Sandbox | `https://sandbox.api.branchapp.example/v1` |
| Production | `https://api.branchapp.example/v1` |

## What the spec covers

* Create and retrieve workers.
* Create payouts and inspect payout status.
* List transfers and card transactions.
* Subscribe to payout and dispute webhook events.
